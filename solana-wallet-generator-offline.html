Generate, import, or vanity-search Solana wallets entirely in your browser. Nothing is sent anywhere — this page makes zero network requests, ever.

Before you fund anything: this runs in a browser tab, which is not a secure place to generate or import a real wallet. Use it to learn, test, or prototype. For a wallet that will hold real funds, use official wallet software (Phantom, Solflare, a hardware wallet) on a device you trust — never a browser tab or an AI chat.

This is not a BIP39 passphrase (the "25th word") — losing it does not lock you out. Your seed phrase alone always recovers this wallet. It only (1) mixes into the randomness used to generate it, as a hedge if you don't fully trust the browser's RNG, and (2) if you download an encrypted backup below, is the password needed to open that file.

Uses base58 characters only — never 0, O, I, or l (they don't appear in Solana addresses).

Decoded and used entirely in this browser tab — never sent anywhere.

Checked against the full BIP39 checksum before use, so a typo gets caught here rather than silently giving you the wrong wallet.

Each additional wallet is a different BIP44 account on the same seed phrase — the same thing Phantom does when you tap "Add / Connect Wallet."
